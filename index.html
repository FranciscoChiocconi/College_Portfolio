About Me

I am currently a sophomore at UC Santa Cruz and have a very deep interest in computer science. I created this website to show others the projects which I have created over my high school and college years and a way to reflect on what I did in the past. I want to pursue computer science in college and am starting to take many steps to make it happen. I have wanted to be a software engineer since I was 14 years old. This career spoke out to me because of the environment which it belongs in. I had a passion for computers, and software engineering is a very interesting branch of the subject. Helping others and creating modern solutions to tricky problems makes this field fun which is why I am pursuing a B.S. in Computer Science.

Projects

Partners J&R Construction

This project is a website that I created while doing freelance web development. I created a customer friendly website for a contractor that included an image carousel, image gallery, and an email inquiry system. The link for this website is partnersjandrconstruction.com and is live right now.

OpenCV2 Lane Detection

I created a computer vision algorithm that is able to detect lines on a road and pedestrians using the computer vision library OpenCV2. I used many of their methods to calculate change in pixel colors and patterns of pixels to help detect lanes.

Weather Web App

This is a website that is available on my Github. It is a web weather app that when given a city as an input it gives statistics about the weather of the city. This used the openweatherapp API to find the data and I used JavaScript to make this data readable.
